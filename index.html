Project 01

湘江智伴自研机器人

2024.08 – 2024.12 · 居家服务机器人 · LLM任务规划 / 通用抓取 / Sim2Real

项目简介

该项目围绕复杂家居场景下的语义理解、自主协作与通用操作能力展开。我主要参与任务规划、灵巧操作与仿真到真机部署相关工作，推动机器人完成收发快递、垃圾分拣等闭环任务。

我的工作

  • 接入大语言模型实现用户意图理解，并通过 Prompt Engineering 与 CoT 方法将高层自然语言指令拆解为可执行的原子动作序列。
  • 主导灵巧手通用抓取与放置算法研发，针对快递、垃圾、办公用品等异形物体优化基于视觉反馈的抓取位姿估计策略。
  • 搭建高保真物理仿真环境进行策略预演与逻辑校验，完成相机内参及手眼标定，提升空间映射精度。
  • 最终实现垃圾分拣、收发快递等复杂任务闭环，真机抓取成功率达到 90% 以上。

建议展示的视频

  • 自然语言指令到任务执行的完整演示
  • 快递或垃圾等异形物体抓取与放置视频
  • 仿真环境与真机执行效果对照视频

Project 02

乐聚机器人

2025.07 – 2025.11 · 居家服务与安全监护 · 跌倒检测 / 抓取感知 / ROS部署

项目简介

该项目聚焦居家服务与安全监护场景，我主导核心视觉算法开发与端到端部署，重点完成跌倒检测、目标识别抓取与感知到控制链路的 ROS 节点化集成。

我的工作

  • 开发基于摄像头的实时跌倒检测系统，利用时空特征提取与关键点追踪技术，有效降低宠物、光影等干扰条件下的误报。
  • 在实机测试中实现 95% 的跌倒检测成功率，提升机器人在安全监护场景中的实用性。
  • 设计轻量化目标检测与分类模型，并结合 RGB-D 点云数据，为机械臂提供精准抓取位姿引导。
  • 在多类目生活用品识别抓取任务中保持 80% 以上成功率，同时将视觉模型高效封装为 ROS 节点，优化推理延迟与系统通讯效率。

建议展示的视频

  • 跌倒检测现场演示视频
  • 多类目物体识别与抓取演示
  • ROS 节点部署后的实机联调演示

Project 03

宇树机器人 G1

2025.12 – 2026.02 · 动作复现与系统集成 · 动捕数据处理 / 仿真训练 / Sim2Real

项目简介

该项目基于 Beyondmimic 开源框架与动捕设备，构建从人体动作采集、数据处理、强化学习训练到宇树 G1 真机动作复现的完整工具链。我重点负责数据工程、轨迹可行性优化及全链路部署测试。

我的工作

  • 制定人体动作采集规范，完成坐标系对齐、数据重采样与格式转换自动化，构建符合宇树 G1 运动学约束的高质量模仿学习数据集。
  • 结合机器人关节限位与动力学特性，对重定向轨迹进行二次校验，剔除物理不可行的高噪帧，提升训练稳定性与收敛速度。
  • 主导策略模型的 Sim2Sim 仿真测试与 Sim2Real 真机部署，打通底层控制与通信链路。
  • 最终在宇树 G1 真机上实现目标动作的成功且稳定复现。

建议展示的视频

  • 仿真训练片段或策略测试视频
